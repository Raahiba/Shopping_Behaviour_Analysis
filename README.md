# Shopping_Behaviour_Analysis
Conducted customer shopping behaviour analysis in Excel using Pivot Tables and Pivot Charts, focusing on parameters like age group, product category, and seasonal trends.
# Tools Used
* Microsoft Excel
* Pivot Tables and Pivot Charts
* Formulas - IFS, COUNTIF, COUNTA
# Insights
This dashboard enables:
* Identification of top-selling products by category
* Analysis of sales performance across seasons
* Viewing of the top 5 cities with the highest and lowest purchase volumes
* Tracking of items purchased by category and size
* Highlighting of the top 5 most purchased and top 5 least purchased items
# Analysis Workflow Summary
Performed data validation by checking for missing values using COUNT and IF functions, confirming that the dataset contained no null entries. Calculated the total number of records and created a new Age Group column using the IFS function to categorize customers for easier analysis.
Developed multiple Pivot Tables to identify top-selling products by category and age group, analyse seasonal sales trends, compare the top 5 cities with highest and lowest purchase volumes, track items purchased by category and size, and highlight the top 5 most and least purchased items. Finally, created Pivot Charts to visually represent all key insights.
